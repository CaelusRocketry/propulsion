CEAgui Download for Microsoft Windows

NASA's CEAgui is a Java GUI Application for NASA's CEA Fortran program. CEAgui creates or modifies your input file (e.g. xxx.inp) and runs a previously complied Fortran executable named FCEA2.exe. FCEA2.exe generates an output file (e.g. xxx.out)on your local machine. The requirements for running this Java Application on your local machine are:

  1. CEAgui JAR file which consists of Java classes, examples, and supporting HTML files.
  2. CEA+ Fortran Package which consists of cea2.f, cea.inc, thermo.inp, trans.inp, cea2.inp, b1b2b3.f, syntax.f.
  3. CEAexec Package which consists of the rest of the supporting files.
  4. If you have not installed the Java2 SDK on your local machine, you must install the Java2 Runtime Environment ( JRE ). JRE is included in the Java2 SDK. You can download the JRE 1.4 executable listed below. Other versions of JRE can be freely downloaded ( download ) from Sun Microsystems Inc.for Microsoft Windows.

The installation procedures for Windows:

  • Create the installation directory on your local machine (e.g. CD C:\CEAexec)
  • Download all three(3)ZIP files and save into the installation directory
    • Extract the CEAgui JAR file (CEAgui-jar.zip)
    • Extract the CEA+ Fortran Package (CEA+Fortran.zip) for CEA files
    • Extract the CEAexec Package (CEAexec-win.zip)
  • If you have not installed the Java2 SDK on your local machine, download JRE 1.4.
    Double-click the executable file (j2re-1_4_1_03-windows-i586.exe) to install JRE 1.4 into the system DEFAULT directory (e.g. C:\Program Files\Java Soft\JRE\1.4)
  • Check that you must have the SIX files (CEAgui.jar, thermo.lib, trans.lib, b1b2b3.exe, syntax.exe, and FCEA2.exe) in the installation directory.
    Double-click the executable batch file CEAexec-win.bat to run the CEAgui program
  • More information can be found in Readme_CEAgui-win.txt
